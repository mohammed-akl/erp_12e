Let users add iFrames to their dashboards

In case you use external tools like Google Analytics, Power BI, Qlik Sense or R-Shiny you might want to give your users a simple way to embed these sites in their personal Odoo dashboard.

To add an iFrame go to Dashboard --> iFrames and create a new entry with

  • Description
  • URL
  • Height

Once created a new iFrame shows up on your dashboard and adjusts to the column layout.
